This Foundation Demo was created by importing CDN links. Enabling this for your own site is a very simple process.

    Add the following line of code to the header of your document (This enables the Foundation CSS): 
                <link href="https://cdn.jsdelivr.net/npm/foundation-sites@6.9.0/dist/css/foundation.min.css" rel="stylesheet">
    Add the following code to the bottom of your body, but still inside it (this enables the JS and imports JQuery, which Foundation uses for many of its functions): 
                <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.6.3/js/foundation.min.js"></script>
            <script>
              $(document).foundation();
            </script>
